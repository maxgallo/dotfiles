# Max's dotfiles
My personal collection of dotfiles.

## Info
For each script, there's a `--remove` option to uninstall that feature

## Order of installation
```bash
./macos.sh
./brew.sh
./git.sh
./tmux.sh
./vim.sh
./fish.sh
./nvm.sh
./iterm.sh
```
